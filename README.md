# Sản phẩm của môn Frontend Framework - học kỳ 5

# Đề tài: Web chat realtime

# Thời gian thực hiện: 19 ngày


## Các chức năng chính
1. Đăng nhập bằng facebook
2. Tìm kiếm người dùng
3. Chat realtime
4. Đánh dấu tin nhắn chưa đọc

---

## Công nghệ sử dụng:
* Backend:
1. Spring MVC
2. Hibernate
3. Websocket
3. Mô hình 3 layer

* Frontend:
1. Angular 7
2. Angular social login
3. StormJS
4. NG Bootstrap
